# Game Assessments

Detailed assessment outputs for individual games, generated by `scripts/game_assessment.py`.

## Structure

```
game_assessments/
└── <game_id>/
    ├── assessment.json    # Structured assessment data
    └── assessment.html    # Human-readable HTML report
```

## Generating

```bash
python scripts/game_assessment.py <game_url_or_id>
```
